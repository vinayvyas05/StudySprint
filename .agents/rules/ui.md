# UI Rules

Build production-quality interfaces.

Prioritize:

- accessibility
- consistency
- responsiveness
- readability

---

## Tailwind

Use consistent spacing.

Avoid arbitrary values unless necessary.

Prefer reusable UI components.

Keep utility classes organized.

---

## React

Prefer:

- functional components
- composition
- custom hooks

Avoid:

- unnecessary state
- prop drilling
- large components

---

## React Native

Follow Expo best practices.

Prefer:

- NativeWind
- SafeAreaView
- optimized FlatLists
- proper keyboard handling

Avoid unnecessary rerenders.

---

Design should feel polished.

Avoid inconsistent spacing, typography and colors.
